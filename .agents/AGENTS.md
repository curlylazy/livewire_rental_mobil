# Custom Agent Rules

- **Model Attributes (PHP 8 Attributes):** Ketika membuat atau memodifikasi Model Eloquent Laravel di direktori `app/Models`, wajib menggunakan PHP Attributes (dari namespace `Illuminate\Database\Eloquent\Attributes`) untuk konfigurasi model, seperti `#[Table]`, `#[Unguarded]`, `#[Fillable]`, `#[Hidden]`, dll., bukan menggunakan properti kelas klasik seperti `protected $table`, `protected $primaryKey`, `protected $keyType`, `public $incrementing`, `protected $guarded`, atau `protected $fillable`.
- **Import Trait & Namespace:** Seluruh import trait (seperti `SoftDeletes`, `HasUuids`, `HasFactory`, dll.) wajib dideklarasikan di bagian atas file menggunakan statement `use` namespace, dan hanya menggunakan nama singkat trait di dalam tubuh class (contoh: `use SoftDeletes, HasUuids;`), bukan menuliskan fully qualified name lengkap dengan backslash di dalam class body (contoh larangan: `use \Illuminate\Database\Eloquent\SoftDeletes;` di dalam class).
