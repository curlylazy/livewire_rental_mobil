- **Objective:** membuat modul master lainnya, sesuai dengan model yang sudah dibuat `app\Models`
- **Goal:** modul yang dibuat mempunya layout yang mirip dengan modul user

## 2. Prepare Skill dan Aturan
- **Skill** : `.agents\skills\form_builder.md` untuk membuat formnya
- **Struktur** : `.struktur.md` struktur data projek
- selalu baca prd.md

## 2. Tahap Pengerjaan
- silahkan kamu baca data modelnya untuk melanjukan membuat modul adminnya
- modul yang kamu akan kerjakan adalah semua yang ada di daftar model `app\Models`
- untuk panduan awal membuat modulnya kamu bisa lihat contohnya di `resources\views\pages\admin\user` modul user
- setelah modul berhasil kamu buatkan routenya sesuai dengan nama modul, disana sudah ada contoh untuk route user
